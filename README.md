FundFlow – Crowdfunding Platform

FundFlow is a web-based crowdfunding platform that connects creators with backers to bring ideas to life. It allows users to create, explore, and support fundraising campaigns through a simple, secure, and user-friendly interface, with real-time tracking of funding progress.

Features
Create and manage fundraising campaigns
Explore and discover active projects
Track funding progress in real time
User authentication (login and signup)
User profile management
Tech Stack

FundFlow is built using HTML, CSS, and JavaScript for the frontend, Python (Flask) for the backend, and SQLite as the database.

Project Structure

The project includes a static folder for assets like CSS and JavaScript, a templates folder for HTML files, the main app.py file for backend logic, and a SQLite database for storing application data.

Setup

Clone the repository, install the required dependencies, and run the Flask application using Python. Once the server starts, open the local host URL in your browser to access the platform.
